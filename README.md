HTML5 实现微信“摇一摇”功能
===

###几个注意点：###

1. 音频需要用户点击加载才可以播放，而且第一次播放有延迟。

2. 全Flex布局，某些浏览器可能不支持。目前支持Chrome、Firefox、Safari、腾讯X5(最新版)。
